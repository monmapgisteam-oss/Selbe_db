# Сэлбэ — Орон зайн мэдээллийн портал

Сэлбэ дэд төвийн GIS портал. Зүүн талын жагсаалтаас **давхарга идэвхжүүлэхэд** тухайн давхаргын дашбоард нээгдэнэ.

Бүх үзүүлэлт ArcGIS FeatureServer-ээс **ажиллах үедээ шууд** татагдана. Апп дотор ямар ч жишиг, демо, зорилтот тоо байхгүй — өгөгдөл татагдаагүй бол тоо харуулахын оронд алдааг ил гаргана.

## Модулиуд

| Модуль | Эх сурвалж | Юу харуулах |
|---|---|---|
| **Ерөнхий мэдээлэл** | `Selbe_talbain_hynalt` | 7 дэд давхарга (барилга, ногоон, зам, явган/дугуйн зам, гэр, гол) — олныг зэрэг харна |
| **Багцын хил** | `bagts_hil` | Төслийн үндсэн хүрээ, багц тус бүрийн явц |
| **Бүсчлэл** | `Busiin_medeelel` | Хот төлөвлөлтийн бүс, FAR/BCR, зогсоолын хүртээмж, төсөв ба гүйцэтгэл |
| **Барилгын явц** | `building_GOL_barigdaj_ehelsen` | Блокийн гүйцэтгэл, 4 түвшин, 16 үе шат, гүйцэтгэгч |
| **Үлдсэн нэгж талбар** | `20260226_uldsen_negj_talbar_selbe` | Чөлөөлөлтийн явц, эзэмшигч, шалтгаан |
| **Газрын үнэ тооцоолуур** | `Selbe_parcel` + `selbe_B` | Талбай зурж, доторх кадастрын талбар ба барилгын үнэлгээ, түрээс, ажлын байр |
| **Шугам сүлжээ ба зам** | `Road_shugam_suljee` | Инженерийн шугамын урт, замын план |
| **Талбайн хяналт** | `survey123_…_results` | Мобайл аппаас ирсэн амьд тайлан, илэрсэн асуудал, байрлалын хяналт |

Аль ч модуль дээр газрын зургийн баруун дээд буланд **«Давхарга нэмэх»** — бусад 17 давхаргын дурыг нь зурагт давхцуулж харна. Эдгээр нь зөвхөн харагдана: дарж сонгогдохгүй, шүүлтэд оролцохгүй, **статистикт нөлөөлөхгүй**.

Төслийн үндсэн хил (`Tuluvlult_talbai` 159.57 га, `Сэлбэ_2_khil`) бүх горимд байнга харагдана.

Дэлгэрэнгүй талбар, тоо, анхаарах зүйлс: [DATA_DICTIONARY.md](DATA_DICTIONARY.md)

## Хөгжүүлэлт

```bash
npm install
npm run dev        # http://localhost:8123
npm run typecheck
npm run build      # static export → out/
```

## Технологи

- **Next.js 15** (App Router) — `output: 'export'` бүрэн статик
- **@arcgis/core 4.31** — газрын зураг. Asset-уудыг CDN-ээс ачаална (`esriConfig.assetsPath`), тиймээс bundle хөнгөн
- **TypeScript**, CSS Modules — гуравдагч UI сан хэрэглэхгүй

ArcGIS SDK нь браузерын API-д (WebGL, ResizeObserver) түшиглэдэг тул порталыг `next/dynamic` + `ssr: false`-ээр ачаална.

### Бүтэц

```
src/
  lib/services.ts     ← бүх ArcGIS URL, талбар, кодын ГАНЦ эх үүсвэр
  lib/query.ts        ← REST асуулга (fetch, POST). Алдааг залгихгүй.
  lib/format.ts       ← тоо, огноо, төгрөгийн формат
  lib/theme.tsx       ← цайвар/харанхуй
  components/         ← Portal бүрхүүл, MapCanvas, UI анхдагчууд
  modules/            ← модуль бүрийн дашбоард
```

Үйлчилгээний URL өөрчлөгдвөл **зөвхөн `src/lib/services.ts`**-ийг засна.

## ArcGIS-ийн анхаарах зүйлс

Дахин бүдрэхээс сэргийлэх гурван зүйл — бүгд бодит алдаа гаргаж байсан:

**Симболын alpha нь төрлөөсөө хамаарна.** Энгийн симбол (`simple-fill`, `simple-line`, `text`) → **0–1**. CIM симбол (`CIMSolidFill`, `CIMSolidStroke`) → **0–100**. Хоёуланг 0–255 гэж бичвэл утга нь тасарч, дүүргэлт бүрэн дүүрэн болно.

**Суурь зураг.** Нэрлэсэн суурь зураг (`gray-vector` гэх мэт) нь **API key шаарддаг** бөгөөд түлхүүргүй үед **чимээгүй** ачаалагдахгүй. Тиймээс нийтийн portal item ашиглана — цайвар `291da5eab3a0412593b66d384379f89f`, харанхуй `5e9b3685f4c24d8781073dd928ebda50`.

**`FeatureLayer.queryExtent()`-ийг битгий ашигла.** Тэр нь `where`-ыг хүсэлтэд огт оруулдаггүй бөгөөд эдгээр FeatureServer түүнийг 400 «No where clause specified» гэж татгалздаг. Хүрээг `src/lib/query.ts`-ийн `queryExtent()`-ээр REST-ээс шууд авна.

Мөн ArcGIS нь `null` ба `' '`-ийг **тусад нь** бүлэглэдэг — `groups()` тэдгээрийг нэгтгэж, `groupWhere()` нь баганад тоологдсонтой яг ижил олонлогийг газрын зурагт шүүнэ.
